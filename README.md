# vs explain

Interactive comparison explanations — learn the difference between commonly confused concepts by playing with them, not reading about them.

**Live site: [tfayyaz.github.io/vs-explain](https://tfayyaz.github.io/vs-explain/)**

## Explanations

- **[quantitative vs qualitative](https://tfayyaz.github.io/vs-explain/quantitative-vs-qualitative/)** — Drag a slider through a day at a coffee shop and watch numbers and words describe the same scene differently.
- **[objective vs subjective](https://tfayyaz.github.io/vs-explain/objective-vs-subjective/)** — Tap words in a sentence to swap between fact and opinion, and see how a single word changes everything.
- **[combination vs permutation](https://tfayyaz.github.io/vs-explain/combination-vs-permutation/)** — Pick coloured balls and see every combination and permutation generated side by side.
- **[trace vs span](https://tfayyaz.github.io/vs-explain/trace-vs-span/)** — Send a request through a microservice system and watch the waterfall timeline build span by span.

## How it works

Each page is a single self-contained HTML file with no dependencies — just vanilla HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Design principles

- No quizzes or testing — learn through interaction, not right/wrong answers
- One strong interactive idea per page
- Mobile and touch friendly
- Minimal serif typography
